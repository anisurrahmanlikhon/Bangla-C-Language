## ইনপুট ও আউটপুট
```

# অন্তর্ভুক্ত < স্ট্যান্ডার্ড_ইন_আউট.হেডার >
পূর্ণ মৌলিক()
{
   পূর্ণ নাম্বার;
   দেখাও ("একটি পূর্ণসংখ্যা নাও");
   নাও("%প", &নাম্বার);
   দেখাও("%প", নাম্বার);
}
```
